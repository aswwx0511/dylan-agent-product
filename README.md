# 火柴特工

Dylan 的网页小游戏作品。

后续 Dylan 使用 Codex 创建的新产品，可以记录在 [`PRODUCTS.md`](PRODUCTS.md)。

## 怎么玩

直接打开 `index.html` 就可以在电脑浏览器里玩。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个公开仓库，比如：`huochai-agent`
2. 把这个文件夹里的文件上传到仓库根目录：
   - `index.html`
   - `style.css`
   - `game.js`
   - `assets/`
   - `README.md`
   - `.nojekyll`
3. 打开仓库的 `Settings` → `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main` 和 `/root`
6. 保存后等待 1 分钟左右

生成的游戏链接通常长这样：

`https://你的GitHub用户名.github.io/huochai-agent/`

只要这个仓库里只放《火柴特工》的文件，就不会公开其他 private 仓库。
